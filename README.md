# 📈 AI Trading Strategy Simulator

## 🚀 Overview

The AI Trading Strategy Simulator is a platform designed to simulate and evaluate trading strategies using historical market data. It aims to help users understand trading performance, risk, and decision-making through data-driven insights.

## 🎯 Objectives

* Simulate stock trading with virtual capital
* Backtest strategies using historical data
* Visualize profit/loss and performance metrics
* Explore AI/ML-based prediction models

## ⚙️ Planned Features

* Technical indicators (RSI, Moving Averages, MACD)
* Strategy backtesting engine
* Profit/Loss visualization dashboard
* Risk analysis (drawdown, volatility)
* AI-based price trend prediction (future scope)

## 🛠️ Tech Stack

* Python
* Pandas & NumPy
* Matplotlib / Plotly
* (Planned) Machine Learning models
